MACFAST Malayalam Keyboard Interface

FAQ

Q)  Which one is suitable for me?

A:    If you are using Windows-98or later, you may download MacfastMalayalam-ASCII.zip If you are using Windows-XP or later MacfastMalayalam-UNICODE.zip. For Linux (Redhat 9.0) systems you can download MacfastMalayalam-rh9-0.1-1.i386.rpm.

Q)  Can I use UNICODE version with Windows-2000?

A:  We do recommend Windows XP or later for UNICODE version, and may cause error with other versions.

Q)  From Where can I get the documentation?

A:  You will get the "How To Install" and user "Manual" from the same page as that of the software.

Q)  Where can I find the font? and how can I install it?

A:  UNICODE fonts can be downloaded from the same page as that of the software. You may unzip the "Font. zip" to your system's 'fonts' folder. Or do as follows.

  1. Extract the Font.zip to any folder.

  2. Open 'My Computer' and then 'Control panel'

  3. Open fonts.

  4. click Files->Install New Fonts.

  5. Select the directory where you have extracted the Fonts.

  6. Select the fonts to be installed from list of fonts and click OK.

  7. Quit and start your application.

Q)   I could install the software and the process is running in system tray. But have no idea how to proceed further. What should I do?

A: The power of our software is that we are not forcing you to use any particular editor.

For ASCII version, you may start the application. Open either MS-Word, Notepad or Paint and set font as ML-TTKartika (May appear in Malayalam letters) and you can type Malayalam.

For UNICODE version, you may start the application and open any application supporting UNICODE like Office XP( Word, Excel, PowerPoint, Outlook, FrontPage ) Outlook Express -6.o, Internet Explorer-6.0, Opera 7.5, WordPad, Paint,  MSN messenger etc.. Set the font of your editor as any UNICODE font (for example Malayalam ). The "Scroll Lock" can be used to switch between Malayalam and English.

For Linux, you may run the application and use Open Office. Set the font of your editor as any UNICODE font (for example Malayalam). If all the locks ("Caps Lock", "Num Lock" and "Scroll Lock") are 'OFF' you will get Malayalam. Otherwise it will be English.

Q)  Will it be possible to use the MACFAST Malayalam fonts by people who doesn't know typing at all and enter text by entering each character
by looking at the keyboard and that too using a standard English keyboard? 

A: If the person who know the pronunciation of each English letters can type Malayalam easily.

Q)  Is there an uninstaller included with these interfaces?

A: In Windows you can uninstall by using 'Add or Remove programs' in control panel.

    In Linux you can uninstall using command 'rpm'.
